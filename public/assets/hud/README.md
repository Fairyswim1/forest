# HUD 에셋

플레이 화면 상단 HUD 프레임용 PNG를 이 폴더에 넣으세요.

| 파일명 | 용도 |
|--------|------|
| `stage_frame.png` | 좌측 스테이지 간판 (나무/돌 프레임) |
| `round_frame.png` | 중앙 Round 둥근 프레임 |
| `score_frame.png` | 우측 점수 패널 |
| `star_icon.png` | 점수 옆 별 아이콘 |
| `menu_button.png` | 메뉴 버튼 배경 (선택) |

텍스트·아이콘(SVG)은 HTML로 얹습니다. `src/types/game.ts` → `ASSETS`
